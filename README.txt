# Midterm Project

My video presentation
https://youtu.be/92u4ILPedhk

Requirements: 
opencv
numpy
numba
matplotlib


1. Source code: seam_carving.py, (optional: image_quality_metrics.py, compare.py, main.py) 
These files should be in the same directory

2. Base images (bench, car, dolphin, and waterfall) should be save in the directory 'images/base/' (the images/ directory should be in the same directory as the source codes)

3. Create a directory call results within the images/directory
    (images/results/)

4. Setup environment 
    PyCharm IDE setup instruction on MAC:
        1. Open PyCharm > Preferences
        2. Expand Project: _ > Python Interpreter
        3. Select [+]  to install available packages
        4. Search: opencv-python3 > Install Package
        5. Search: numpy > Install Package
        6. Search: numba > Install Package
        7. Search: matplotlib > Install Package
        8. Click OK
        9. Open main.py
        10. Run main
    Or install libraries and run python using command line terminal:
        pip install opencv-python
        pip install numpy
        pip install numba
        pip install matplotlib
        python3 main.py

5. (Optional) Run compare.py to see the image comparison metrics
    The methods used in compare.py are defined in image_quality_metrics.py
